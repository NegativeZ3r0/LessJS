## Basic counter app

### Using crazy framework stack 

- [facil.io](http://facil.io) a minimal backend framework for C
- [HTMX](https://htmx.org/docs/) a html first ui framework which follows HATEOAS principle

### How to use?

Dependancies:
- [gcc](https://gcc.gnu.org/)
- [make](https://www.gnu.org/software/make/)
- [npm](https://docs.npmjs.com/about-npm)

just do:

```bash
git clone https://github.com/NegativeZ3r0/LessJS.git
cd LessJS
npm install
npm run dev
```
(ignore the warnings ;)

then go on http://localhost:3000
